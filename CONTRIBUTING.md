# Contributing to ai.txt

## Adding Your Site to the Directory

1. Fork this repository
2. Add your domain to [SITES.md](SITES.md) in the appropriate category
3. Ensure your site has a valid `/ai.txt` file accessible via HTTPS
4. Submit a pull request

## Proposing Specification Changes

For changes to the ai.txt specification:

1. Open an issue describing the proposed change
2. Include rationale and examples
3. Allow for community discussion before submitting a PR

## Guidelines for ai.txt Content

When creating your own ai.txt file:

- Keep it plain text with markdown headings for structure
- Include only factual, current information
- Update it when your business details change
- Aim for completeness but avoid unnecessary verbosity
