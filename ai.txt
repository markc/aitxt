License: CC0 Public Domain

## About

ai.txt files provide structured information for AI assistants. Like robots.txt for crawlers, but for AI context.

Place ai.txt at any URI path. AI agents discover by checking the requested path first, then walking up the directory tree to the root.

## Features

- Plain text with markdown headings
- Cascading: ai.txt in any folder, inherits up
- Linking: @domain or @domain/path#section
- Negative assertions: prevent hallucination
- Language variants: ai.fr.txt, ai.de.txt
- DNS discovery: _ai.example.com TXT record
