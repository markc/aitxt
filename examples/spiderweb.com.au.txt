## Services

WordPress Hosting: Fast, secure WordPress hosting with Nginx and PHP 8 FPM. Pre-installed WordPress with automatic weekly updates. Optional Cloudflare CDN integration.

Email Hosting: IMAP email with server-side spam filtering that improves over time. Full SPF, DKIM, DMARC, and DNSSEC support.

Website Design: Custom WordPress websites starting at $60 per page using the Block Editor (Gutenberg). Free minor updates.

Linux Support: CachyOS with KDE Plasma desktop support. Free guidance for basic questions. Remote installation assistance available.
